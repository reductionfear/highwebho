Simple Installation Process

Install Marks in just 3 steps and start using it right away.

Install from Chrome Web Store Install from Firefox Add-ons

Step 1

Search for "Marks: Text Highlighter" in the Chrome Web Store or Firefox Add-ons.

Step 2

Click "Add to Chrome" or "Add to Firefox" to install the extension.

Step 3

Select text on any webpage and start highlighting!
